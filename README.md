### Installation :

```bash
pkg update && pkg upgrade -y
pkg install git -y
pkg install python ruby -y
gem install lolcat
git clone https://github.com/ShTasrif/NumberToInfo
pip install requests
cd NumberToInfo
python info.py
```
### Run Tools

```bash
numbertoinfo
```
